### <a id="TLocation"></a>TLocation

__Type__  
TLocation = TPoint;  
__Description__  
The TLocation type is used to define a point in X,Y coordinates for a design object\.  
Where the TPoint = packed record  X: Longint;  Y: Longint;end;
