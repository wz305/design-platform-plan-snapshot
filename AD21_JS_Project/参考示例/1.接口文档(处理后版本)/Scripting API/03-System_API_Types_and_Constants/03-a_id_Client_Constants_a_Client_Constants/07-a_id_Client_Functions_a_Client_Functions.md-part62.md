#### ExtendedToEng

__Declaration__  
Function  ExtendedToEng \(Const ExtVal    : Extended\) : String;  
__Description __  
The ExtendedToEng function converts the floating\-point value given by Value to its string representation\.   
__Example__  
ShowInfo\(ExtendedToEng\(4\.32e18\)\); //4\.320e18
