#### IntSwap

__Declaration__  
Procedure IntSwap\(Var a,b : Integer\);  
__Description __  
The IntSwap procedure swaps the values for A and B\. For example A = 2 and B = 5\. After passing these values into IntSwap procedure, the new values are a = 5 and b = 2\.
