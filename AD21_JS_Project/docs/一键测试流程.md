一键测试流程（Spec v0.1）

Step 1：运行 `scripts/一键启动_最小闭环.cmd`（启动 Node Mock + 打开浏览器 UI）
Step 2：打开 AD，打开 UI 面板
Step 3：点击按钮“Spec0.1 一键验证”（或运行 `测试_AD_Spec_0_1_一键验证()`）

期望 UILogger 关键日志关键字：PING / COMMAND / BOARD_SUMMARY / UPLOAD_REPORT / FINAL_SUMMARY

补充（AD 侧验证点）

- BOARD_SUMMARY 应包含有效 name/units/bounds，以及 counts(track/via/pad/total) 非零
- 若 counts 为 0 或缺失，优先关注 BOARD_SUMMARY_COUNT_DIAG
